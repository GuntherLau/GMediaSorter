## 功能特性

- 📁 浏览本地文件夹中的视频文件
- 🎬 支持多种视频格式（MP4, AVI, MKV, MOV, WMV, FLV, WebM, M4V）
- 📊 显示视频文件详细信息（大小、修改日期、格式、分辨率、时长）
- ✅ 多选视频文件
- 🎯 **多维度过滤**：支持分辨率、时长与长宽比组合过滤
  - **分辨率过滤**：<720p / 720p / 1080p / >1080p
  - **时长过滤**：30秒内 / 30秒-2分钟 / 2-10分钟 / 10-30分钟 / 30分钟-1小时 / 超过1小时
  - **长宽比过滤**：竖屏 / 方形 / 标准横屏 / 超宽屏 / 仅查看未知（快捷定位缺失元数据）
  - 可折叠过滤面板，一键清除所有过滤，并支持“仅查看未知”快速切换
- 🔍 **重复检测**：基于哈希的完全重复文件检测，释放存储空间
- 🎯 **相似检测**：基于视频指纹的内容相似度分析，智能发现相似视频
- 🗑️ 批量删除重复和相似文件
- �️ 批量视频编码转换（H.264 / H.265），支持进度跟踪、取消与结果摘要
- 📦 批量容器封装转换（MP4 / MKV），无损调整封装格式，已符合目标容器时自动走快速复制
- ▶️ 内置视频播放器，支持播放控制、倍速、音量、循环、全屏与元数据展示
- �🎨 现代化的用户界面

## 安装依赖

```bash
npm install
```

### 系统依赖（仅开发环境）

开发时需要安装 ffmpeg：

**macOS:**
```bash
brew install ffmpeg
```

**Windows:**
```bash
choco install ffmpeg
```

**Linux:**
```bash
sudo apt-get install ffmpeg
```

> **注意**: 打包后的应用会自动包含 ffmpeg,最终用户无需手动安装。详见 [部署指南](docs/DEPLOYMENT.md)。

## 开发模式

运行开发服务器和 Electron 应用：

```bash
npm run electron:dev
```

或者分别运行：

```bash
# 终端 1: 启动 Vite 开发服务器
npm run dev

# 终端 2: 编译 Electron 代码并启动应用
npm run build:electron && electron .
```

运行容器转换服务的单元测试：

构建生产版本：

```bash
npm run electron:build
```

生成的应用将位于 `release` 目录。

## 使用说明

### 基本操作

1. 启动应用后，点击"选择文件夹"按钮
2. 选择包含视频文件的文件夹
3. 应用将自动扫描并显示所有支持的视频文件（包含分辨率和时长信息）
4. 点击视频卡片可以选中/取消选中文件

### 多维度过滤

应用提供强大的多维度过滤功能，可以同时按分辨率、时长与长宽比筛选视频：

#### 分辨率过滤
- **< 720p**：标清视频
- **720p**：高清视频
- **1080p**：全高清视频
- **> 1080p**：2K/4K及以上

#### 时长过滤（互斥区间）
- **30秒内**：超短视频/片段
- **30秒-2分钟**：短视频/音乐MV
- **2-10分钟**：中短视频/预告片
- **10-30分钟**：中长视频/剧集
- **30分钟-1小时**：长视频/电影
- **超过1小时**：超长视频/直播回放

#### 长宽比过滤（互斥区间）
- **竖屏**：典型 9:16 / 3:4 内容
- **方形**：接近 1:1 的画面
- **标准横屏**：覆盖 4:3、3:2、16:9
- **宽屏/电影**：包含 21:9、2.35:1 等超宽画面
- **仅查看未知**：快捷筛选缺失宽高信息的素材，再次点击可恢复上一档位

**使用技巧**：
- 可同时选择分辨率、时长与长宽比进行组合过滤
- 点击过滤器面板顶部的折叠按钮可收起/展开面板
- 使用"清除全部"按钮一键重置所有过滤条件
- 当存在未知长宽比素材时，摘要会提示“含 X 个未知长宽比”
- 过滤结果会实时显示"已过滤 X / Y"

### 重复检测（找相同）

1. 扫描视频后，点击工具栏的"🔍 找相同"按钮
2. 应用将进行两阶段检测：快速预筛选 → 精确验证
3. 检测完成后显示重复文件组，包括：
   - 每组的重复文件列表
   - 总占用空间和浪费空间统计
   - 智能标记最早的文件为保留项
4. 选择要删除的重复文件，点击"删除选中"按钮批量删除

### 相似检测（找相似）

1. 扫描视频后，点击工具栏的"🎯 找相似"按钮
2. 应用将进行三级检测：元数据筛选 → 视频指纹提取 → 相似度计算
3. 检测完成后显示相似视频组，包括：
   - 相似文件组及平均相似度
   - 详细的相似度评分（综合/视觉/时长/分辨率/大小）
   - 两两配对的相似度详情
4. 根据需要手动处理相似视频

### 视频编码转换

1. 勾选要转换的一个或多个视频
2. 在系统菜单中选择 `工具 → 视频编码转换`，展开下拉菜单
3. 选择目标编码（H.264 或 H.265）后，指定输出目录（支持新建文件夹）
4. 查看进度对话框：实时显示转换进度、成功/失败计数和当前文件名
5. 可随时点击“取消转换”终止队列，已完成的文件将保留在输出目录
6. 转换完成后弹出结果摘要，可直接打开输出目录或查看失败原因

### 容器格式转换

1. 勾选要处理的一个或多个视频（默认处理当前过滤列表中的全部文件）
2. 在工具栏点击“容器转换”或通过菜单 `工具 → 容器格式转换`
3. 选择目标封装容器（MP4 或 MKV），确认后指定输出目录
4. 查看进度对话框：实时显示处理数量、当前文件和目标容器
5. 若源文件已符合目标容器，将跳过 remux 执行快速复制
6. 可随时点击“取消转换”停止后续队列，已完成的文件保留
7. 转换结束后弹出摘要，可直接打开输出目录或查看容器转换日志

### 视频预览/播放

1. 在任意列表或检测结果中点击视频卡片上的“播放”按钮
2. 播放器侧边面板会显示视频名称、路径和编解码/分辨率/码率等基础信息
3. 使用底部控制区调整播放/暂停、静音、音量、倍速（0.5x~2x）、循环模式
4. 空格键切换播放，左右方向键微调进度，F 键进入/退出全屏，Esc 或关闭按钮收起面板
5. 播放器会记住最近一次的自动播放、循环和倍速偏好，可在设置区域调整

## 常见问题

### Q: 为什么需要安装 ffmpeg？

A: ffmpeg 用于视频帧提取和视频指纹计算。开发环境需要安装系统级 ffmpeg,但打包后的应用会自动包含 ffmpeg,用户无需手动安装。详见 [FFmpeg 打包方案](docs/FFMPEG-PACKAGING.md)。

### Q: 如何验证 ffmpeg 配置是否正确？

A: 运行 `./verify-ffmpeg.sh` 脚本,它会检查所有 ffmpeg 相关配置。

### Q: 应用支持哪些视频格式？

A: 支持常见的视频格式:MP4, AVI, MKV, MOV, WMV, FLV, WebM, M4V 等。

### Q: 检测结果会保存吗？

A: 目前不会自动保存。未来计划添加结果缓存和导出功能。
